# Exsa-Engine Documentation

This directory contains important documentation and analysis files for the Exsa-Engine project.

## Files

- `BEAST_MODE_VERIFICATION.md` - Complete beast mode performance verification
- `PERFORMANCE_METRICS.md` - Detailed performance metrics and benchmarks
- `BUILD_SCRIPT_EXPLAINED.md` - Explanation of the universal build script
- `LFM2_ENGINE_ANALYSIS.txt` - Analysis of LFM2 model performance with this engine
- `ENGINE_UNIVERSALITY_EXPLAINED.txt` - Explanation of universal GGUF model support
- `COMPETITIVE_ANALYSIS.md` - Comparison with other inference engines
